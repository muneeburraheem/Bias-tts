This repository contains all code, evaluation scripts, figures, datasets, and the LaTeX source for the paper:

**"Mitigating Social Bias in English and Urdu Language Models Using PRM-Guided Candidate Selection and Sequential Refinement"**

The project introduces a unified *inference-time* debiasing pipeline for single-word completions in English and Urdu using:

- **Baseline** (raw GPT output)  
- **PRM-Select** (Best-of-N sampling with bias+utility scoring)  
- **PRM-Sequential** (iterative refinement using PRM critiques)

The repository includes an end-to-end evaluation pipeline, generation scripts, PRM scoring utilities, bilingual prompt dataset (400 items), and reproducible figures/tables for research publication.
